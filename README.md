# AI Director Pro - 智能运镜系统

AI Director Pro 是一款基于 Electron 和 TensorFlow.js 开发的专业级智能运镜系统。它利用先进的 AI 人脸识别技术，实现对人物的自动跟踪与拍摄，专为直播、在线会议及视频录制场景设计。

## ✨ 核心特性

### 🎥 智能运镜与追踪
- **高精度识别**：基于 Google BlazeFace 模型，提供毫秒级的人脸检测。
- **平滑运镜**：内置平滑算法，确保镜头移动自然流畅，避免抖动。
- **自定义参数**：支持调节缩放比例、跟随死区、平滑度等参数，满足不同场景需求。
- **多摄支持**：支持系统内多个摄像头设备的无缝切换。

### 🚀 性能与体验优化
- **全速运行模式**：强制应用在前台及后台均全速运行，防止系统休眠或降频导致的画面卡顿。
- **智能召回机制**：
  - 支持 **隐身模式**（将窗口移至屏幕外，不影响 OBS 推流）。
  - 具备 **智能召回** 功能：点击任务栏图标或窗口意外移出屏幕时，自动将窗口恢复至视野中心。
- **安全退出保护**：提供关闭确认对话框，防止直播过程中因误触导致程序退出。

### 🎨 画面增强与控制
- **美颜滤镜**：支持实时磨皮与亮度调节，提升出镜效果。
- **画面调整**：支持画面 90° 旋转及水平镜像翻转。
- **窗口管理**：
  - **窗口置顶**：确保监控画面始终可见。
  - **比例锁定**：一键锁定窗口宽高比。
  - **极简模式**：支持隐藏 UI 仅保留画面。

### 💾 配置与扩展
- **参数持久化**：自动保存用户配置。
- **配置导入/导出**：支持 JSON 格式的配置文件导入导出，方便在不同设备间同步设置。

## 🛠️ 技术栈

- **核心框架**: [Electron](https://www.electronjs.org/) (v34.5.8)
- **AI 引擎**: [TensorFlow.js](https://www.tensorflow.org/js) (v3.21.0)
- **人脸模型**: [@tensorflow-models/blazeface](https://github.com/tensorflow/tfjs-models/tree/master/blazeface) (v0.0.7)
- **打包工具**: [electron-builder](https://www.electron.build/) (v24.13.3)

## 📦 安装与构建

### 前置要求
- Node.js (建议 v16 或更高版本)
- npm 或 yarn

### 开发环境设置

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd AI-Director
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发模式**
   ```bash
   npm start
   ```

### 打包发布

将项目打包为可执行文件（Windows .exe）：

```bash
npm run build
```

打包完成后，安装包将生成在 `build_output` 目录下。

## 🎮 使用指南

### 快捷键
- **Ctrl + Alt + H**: 切换 **隐身模式**（隐藏/显示窗口）。

### 界面操作
1. **启动**: 点击界面上的 "🚀 启动" 按钮初始化摄像头和 AI 模型。
2. **调参**: 点击 "⚙️ 调参" 展开设置面板，调整平滑度、缩放、美颜等参数。
3. **锁定**: 鼠标移至屏幕右侧边缘，点击锁定按钮可防止误触界面控件。
