# AI Director Pro - 智能运镜系统 (v2.0.0)

欢迎使用 **AI Director Pro**！这是一款专为直播、会议和视频录制设计的智能运镜助手。它能通过 AI 自动识别人脸，控制镜头跟随，让你的画面始终聚焦于你。

## 📥 下载与安装

### 方式一：直接下载（推荐）
本仓库包含最新版本的安装包。
请直接点击上方文件列表中的 **[AI Director Pro Setup 2.0.0.exe](AI%20Director%20Pro%20Setup%202.0.0.exe)** 进行下载。

### 方式二：Releases 页面
你也可以点击页面右侧的 **Releases** 查看和下载历史版本。

### � 安装步骤
1. 下载完成后，双击运行 `AI Director Pro Setup 2.0.0.exe`。
2. 按照安装向导提示完成安装（通常只需点击"下一步"）。
3. 安装完成后，桌面会出现 **AI Director** 图标，双击即可运行。

## 🚀 快速开始

1. **连接摄像头**：确保你的电脑已连接摄像头。
2. **启动软件**：打开软件，点击界面中央的 **"🚀 启动"** 按钮。
3. **开始运镜**：软件会自动检测人脸并开始追踪。你可以自由移动，镜头会跟随你。

## ✨ 主要功能

*   **🎥 智能追踪**：AI 自动识别人脸，平滑跟随，拒绝画面抖动。
*   **💄 实时美颜**：内置磨皮与亮度调节，让你时刻保持最佳状态。
*   **🛡️ 隐身模式**：支持将窗口移出屏幕外（配合 OBS 使用），不占用桌面空间。
    *   *快捷键*：`Ctrl + Alt + H` 切换显示/隐藏。
    *   *智能召回*：点击任务栏图标或将鼠标移出屏幕边缘可触发召回机制。
*   **⚙️ 专业调参**：可自定义缩放比例、跟随速度、画面镜像及旋转。

## ⚠️ 注意事项与常见问题

### 1. 摄像头权限
首次运行时，系统可能会询问摄像头访问权限，请务必选择 **"允许"**。如果画面黑屏，请检查 Windows 设置 -> 隐私 -> 摄像头中是否允许了应用访问。

### 2. 配合 OBS 直播使用
*   在 OBS 中添加 **"窗口采集"** 源。
*   选择 `[AI Director Pro]` 窗口。
*   **技巧**：你可以使用软件的"隐身模式"将窗口移出屏幕可视区域，OBS 依然可以采集到画面，这样不会遮挡你的桌面操作。

### 3. 性能建议
本软件使用 GPU 加速 AI 运算。如果遇到卡顿，请尝试：
*   关闭其他占用显卡的大型程序。
*   确保电脑连接了电源（笔记本模式下）。
